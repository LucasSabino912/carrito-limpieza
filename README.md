1. Levantar el Backend (FastAPI / Python)

    Activar el entorno virtual (venv):

        source venv/bin/activate

    Correr el servidor:

        uvicorn main:app --reload
        (Se va a levantar en [http://127.0.0.1:8000](http://127.0.0.1:8000))

2. Levantar el Frontend (Next.js / React)

    Correr el servidor de desarrollo:

        npm run dev
        (Se va a levantar en http://localhost:3000)
